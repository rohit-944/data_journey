## 📅 WEEK-BY-WEEK PLAN (26 Weeks)

### ═══════════════════════════════════════
### MONTH 1: SQL + PYTHON FOUNDATION (Weeks 1-4)
### ═══════════════════════════════════════

**Theme:** The 2 skills ALL 3 roles demand — SQL and Python

#### Week 1 (June 16-22)
- Install PostgreSQL + DBeaver (free GUI)
- SQL: SELECT, WHERE, GROUP BY, HAVING, ORDER BY, LIMIT
- JOINs: INNER, LEFT, RIGHT, FULL OUTER, SELF JOIN
- Practice: 20 HackerRank SQL Easy + 5 LeetCode SQL Easy
- DataPathSala: 2 SQL problems/day
- Watch: Ankit Bansal YouTube — "SQL for Interviews" (first 5 videos)
- **Time:** 6-8 hours/day

#### Week 2 (June 23-29)
- Window Functions: ROW_NUMBER, RANK, DENSE_RANK, NTILE, LAG, LEAD, PARTITION BY
- Running totals, moving averages, cumulative sums
- CTEs (Common Table Expressions), recursive CTEs
- Subqueries: correlated vs non-correlated
- CASE WHEN, COALESCE, date functions, string functions
- Practice: 5 LeetCode SQL Medium/day + 3 DataPathSala/day
- StrataScratch: Start solving real company questions (free tier)

#### Week 3 (June 29 - July 5)
- Install Anaconda + Jupyter Notebook
- Pandas: DataFrames, read_csv, filtering, groupby, merge, pivot_table, apply, melt
- NumPy: Arrays, reshaping, statistical operations
- Data Cleaning: nulls, duplicates, type conversion, outliers, string cleaning
- Complete: Kaggle "Pandas" micro-course (free, 4 hours)
- Hands-on: Download 3 Kaggle datasets, clean them completely
- Continue: 3 SQL problems/day (NEVER stop SQL)

#### Week 4 (July 6-12)
- Visualization: Matplotlib + Seaborn (bar, line, scatter, histogram, heatmap, boxplot)
- Statistics: Mean, median, mode, std dev, percentiles, correlation
- Probability basics, normal distribution, p-value, hypothesis testing, A/B test concept
- Excel: Pivot tables, VLOOKUP/XLOOKUP, IF/COUNTIF/SUMIF
- Google Sheets: Same formulas
- Continue: 3 SQL + 2 Pandas problems/day

**✅ Month 1 Checkpoint:** SQL interview-ready + Pandas fluent + Statistics basics done

**Theme:** Power BI (for DA) + dbt (for AE) + Spark basics (for DE) — cover all 3

#### Week 8 (Aug 3-9)

- Hands-on: Upload file to S3, query with Athena
- Docker basics: Dockerfile, docker-compose, containerize a Python script
- Resource: Marc Lamberti YouTube (Airflow), DataTalks.Club DE Zoomcamp
- Continue: 3 SQL + 2 PySpark problems/day
